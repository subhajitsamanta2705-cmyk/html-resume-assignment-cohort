## Project Description

This project is a simple **HTML resume** created using **semantic HTML5**.  

### Structure Overview

- **Header**
  - Introduction section with name and professional title
  - “About Me” section describing background and experience

- **Main Content**
  - **Skills Section:** Unordered list showcasing technical skills
  - **Experience Section:** Table displaying job roles, companies, and durations
  - **Projects Section:** Articles describing completed projects, technologies used, and timelines
  - **Education Section:** Table outlining academic qualifications
  - **Contact Section:** Basic contact information including email, phone, and location

- **Footer**
  - Copyright notice

### HTML Concepts Demonstrated


- Semantic elements (`header`, `main`, `section`, `article`, `footer`)
- Headings (`h1`, `h2`)
- Lists (`ul`, `li`)
- Tables (`table`, `tr`, `th`, `td`)
- Text formatting (`p`, `strong`)
- Proper document structure and readability
